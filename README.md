## What is fpAssist?

fpAssist is a desktop **AFK companion app for Fishing Planet** built for players who want repetitive lure fishing to be smoother, faster, and easier to manage.

Instead of redoing the same cast/retrieve/reel flow manually every session, fpAssist gives you:

- a compact in-game control bar
- profile-based fishing behavior
- quick hotkey access to common actions
- easier stop/start handling for repeated runs
- support for AFK-friendly lure fishing loops

---

## What it’s meant for

fpAssist is especially useful for players who spend a lot of time doing:

- repeated lure casting
- stop-and-go retrieval
- twitch-style retrieval
- long grind sessions
- semi-AFK fishing loops
- setup tuning for one favorite method

It is not just a random key macro. It is meant to be something you can actually use as part of your regular Fishing Planet routine.

---

## What stands out

### A real in-game control surface
The overlay bar makes the app feel practical instead of hidden. You can see what profile you’re on, access core actions quickly, and manage your session without digging around.

### Profile-based fishing
One of the most useful parts is being able to tune and reuse fishing profiles instead of rebuilding the same behavior every time.

### AFK-first mindset
A lot of players searching for something like this are really searching for a way to make repetitive gameplay less tedious. fpAssist clearly leans into that use case.

### Built for repeated lure loops
The strongest fit seems to be lure fishing where you want consistency in cast distance, sink time, reel rhythm, twitch behavior, or stop-and-go pacing.
